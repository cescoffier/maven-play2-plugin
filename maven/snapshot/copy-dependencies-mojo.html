play2:copy-dependencies

Full name:

de.akquinet.innovation.play2:play2-maven-plugin:1.2.2-SNAPSHOT:copy-dependencies

Description:

Copy project dependencies to the lib folder.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.

Optional Parameters

Name Type Since Description
lib File - Where are the dependencies copied.
Default value is: lib.

Parameter Details

lib:

Where are the dependencies copied.
  • Type: java.io.File
  • Required: No
  • Default: lib
